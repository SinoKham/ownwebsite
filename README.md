Проект - сайт-визитка на 5 страниц: главная, обо мне, навыки, портфолио, контакты. Сделан по трём критериям.

Первое - адаптив. Mobile-first подход, под планшеты и десктоп меняется меню и колонки карточек. Есть тёмная и светлая тема, выбор сохраняется в localStorage.

Второе - интерактив. Калькулятор на странице навыков, конвертер валют на странице обо мне, приветствие по времени суток в шапке, счётчик посещений в углу. Форма обратной связи с валидацией фио (только буквы), телефона, email, даты связи и загрузкой фото.

Третье - архитектура. Всё разбито на модули. BasePage для общих методов, отдельные компоненты ProjectCard и DataTable с фильтрацией и сортировкой. Все данные вынесены в constants.js, вспомогательные функции в helpers.js (работа с массивами, объектами, строками, датами, валидация).
